

# Colyseus
1. Server
2. Client
    * Unity

## Installation

  ### Prerequisites
  - Git 
  - Node 5.0 or upper

#### Install in your own project

copy `Assets/Plugins` and `Server` to your project then

```bash
$ cd Server
$ npm install
$ npm start
```
or, 

#### Edit the example project

```bash
$ git clone https://github.com/IsuraManchanayake/colyseus-unity3d.git
$ cd colyseus-unity3d/Server/
$ npm install
$ npm start
```
